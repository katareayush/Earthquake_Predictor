About the Project

Project Overview

Our project is based on an AI/ML model that uses a dataset of over 3000 past earthquakes to train the ML model and AI to make highly accurate predictions on any future calamities. Based on these predictions, a push button is activated sending a universally accepted Morse code SOS signal to ensure that help arrives in a timely and efficient manner Our revolutionary project can save millions of lives in the future by aiding rescue operations and providing help as quickly as possible.

How It Works

Our model uses the very accurate and renowned machine learning model, RandomForest. It creates several data trees and conducts Bootstrap sampling on each one of them to create an aggregate result. This result is as accurate as possible ensuring a trustworthy output. We have then created our UI using Flask and HTML/CSS to ensure that it is concise, thereby displaying all important information in a clear manner, yet attractive. This ensures that our project is as accurate as possible, along with having a user friendly UI.
